Syntax Highlight and snippets for XV2 PQ modding.

Most Super/Ult attacks have a snippet as well as evasives and awkenings

NOTE: some skills snippets might not have a comment next to them.


INSTALLATION:

- Go to extension tab on VSC
- In top right of tab click the three dots (...)
- Select "Install From VSIX"
- Select the x2qs.vsix file
- Done!






NOTE: All extension files are in repo for modification if you see fit!
